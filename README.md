 🤖 Tesla Bot Animated Page

Página animada inspirada no Tesla Bot, desenvolvida para praticar animações com **GSAP** e construção de layouts modernos em front-end.

<hr>

🚀 Tecnologias
- HTML5
- CSS3
- JavaScript
- GSAP

<hr>

 🎯 Objetivo
Praticar estruturação de código e aplicação de animações, transições e sequenciamento de elementos para criar uma experiência visual mais dinâmica.

<hr>

 🙌 Créditos
Projeto aplicado com base nos ensinamentos do **CodeBoost**.
